# Analysis Plan Template

> **Generalized template. Client‑specific versions are stored privately and covered by NDA.**

## 1. Purpose
This analysis plan template provides a structured framework for quantitative and qualitative evaluation activities. It is suitable for RCTs, quasi‑experimental studies (PSM, DiD), process evaluations, and mixed‑methods designs.

---

## 2. Evaluation Questions (Examples)
1. What is the program’s effect on primary outcomes (e.g., learning, adoption, welfare)?
2. What are the intermediate changes in knowledge, attitudes, practices?
3. What contextual factors influence program effectiveness?
4. What mechanisms explain observed effects?

---

## 3. Data Sources
- **Primary data**: Structured surveys, KIIs, FGDs, observations  
- **Secondary data**: Administrative records, monitoring datasets, test scores  
- **Metadata**: Sampling frames, enumerator logs, consent forms  

---

## 4. Outcomes & Indicators

### 4.1 Primary outcomes
- Continuous: test scores, income, indices  
- Binary: enrollment, adoption, retention  
- Ordinal: Likert‑scale responses  

### 4.2 Intermediate outcomes
- Knowledge, attitudes, exposure, participation intensity  

### 4.3 Covariates
- Demographics  
- Baseline performance  
- Household characteristics  
- Location fixed effects  

---

## 5. Statistical Methods

### 5.1 Descriptive analysis
- Means, medians, percentiles  
- Cross‑tabulations  
- Weighted estimates (if applicable)

### 5.2 PSM estimation
- Model: logit/probit  
- Matching: nearest‑neighbor, kernel, caliper  
- Balance checks: SMDs, density plots, variance ratios
