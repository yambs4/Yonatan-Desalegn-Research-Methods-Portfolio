# My Portfolio

In this repo, I present a portfolio showcasing selected evaluation projects. Reproducible code (Stata, Python/Jupyter, R), data-processing pipelines, and documentation for tracer studies, endline evaluations and governance/food-security impact work — with annotated replication scripts, result tables, and methodological notes.

I have selected three of the most relevant projects I have worked on:

---

## 1) Speed School Tracer Study
Repository: https://github.com/yambs4/SSP_tracer_study (private repo)

Short description
- Replication and analysis code for the Speed School Program (SSP) tracer study and impact evaluation in Afar (Amibara and Asaita) and Somali (Babile) regions of Ethiopia.

Key methods & tools
- Study design replication, survey data cleaning, descriptive statistics, impact estimation (matching / regression)
- Main languages: Stata (primary), Python (support), R (occasional)

My contributions (end-to-end 
- Prepared the inception report, designed the evaluation methodology, prepared survey instruments, deployed field instruments via Kobo Toolbox
- Conducted the replication of baseline and endline analyses and assembled reproducible scripts for dataset cleaning and analysis.
- Implemented robust standard errors and cluster adjustments for the field sampling design.
- Produced reproducible tables and visualizations used in the final report.
- Drafted the main report.

## 2) The Leprosy Mission International - Ethiopia (TLMIE) project endline evaluation
Repository: https://github.com/yambs4/TLMIE_endline_eval (private repo)

Short description
- Code and data repo for the impact evaluation of community-based training and disability inclusion facilitators methodology on governance and food security.

Key methods & tools
- Program evaluation methods, pre-post / cross-sectional impact analysis, indicator construction, program monitoring metrics
- Main language: Stata

My contributions
- Lead the endline evaluation of the project end-to-end.
- Wrote end-to-end Stata do-files to reproduce the analysis pipeline (raw → cleaned → analysis → tables).
- Constructed composite indicators for governance and food security and validated them with sensitivity checks.
- Prepared data documentation (data dictionary + codebook) to ensure reproducibility for stakeholders.
